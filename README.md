# AI Image Generator

A responsive web application that allows users to generate AI-powered images based on text prompts. Users can select different AI models, specify the number of images to generate, and choose the desired aspect ratio. The app supports light and dark themes with smooth toggling.

---

## Features

- **Prompt Input:** Describe your imagination in detail to generate images.
- **Random Prompt Generator:** Quickly fill the prompt with example creative ideas.
- **Model Selection:** Choose from multiple AI image generation models.
- **Image Count:** Generate 1 to 4 images at a time.
- **Aspect Ratio:** Choose between Square (1:1), Landscape (16:9), and Portrait (9:16).
- **Image Gallery:** Displays generated images with download option.
- **Theme Toggle:** Switch between light and dark modes, with preferences saved.
- **Responsive Layout:** Clean, user-friendly design with modern UI components.
- **Loading and Error Handling:** Visual feedback during image generation and error messages on failure.
